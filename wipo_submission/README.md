# WIPO Submission Files

This folder contains a clean Overleaf-ready draft for the WIPO Regional Seminar on Text and Data Mining for Universities and Research-Oriented Institutions in Africa.

## Files

- `main.tex`: single-file LaTeX draft containing the title, co-authored abstract, and brief biographies.

## Suggested usage

1. Create a new Overleaf project.
2. Upload `main.tex`.
3. Compile and export as PDF.
4. If needed, copy the final text into Word for submission.
